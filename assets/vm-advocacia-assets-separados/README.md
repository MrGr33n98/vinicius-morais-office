# Pacote de Assets — VM Advocacia

Este pacote contém os elementos visuais separados a partir da landing page aprovada.

## Formatos
- PNG: máxima compatibilidade e qualidade
- WebP: uso recomendado no Next.js
- JPG: alternativa leve para fotografias

## Uso recomendado
- Hero: 01-hero-cliente-area-do-cliente
- Seção SaaS/Área do Cliente: 02, 03 e 04
- Atendimento/rodapé: 05
- Blog: 06 a 08
- SEO local/cidades: 09 a 15

## Observação
Os arquivos foram extraídos do mockup aprovado e ampliados com reamostragem de alta qualidade.
Como a origem é uma composição de página, alguns recortes podem conter elementos da interface.
